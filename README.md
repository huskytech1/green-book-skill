# Green Book (智造三点三小绿书)

智造三点三（Zhi Zao San Dian San）品牌专用小绿书新闻生图工具。

## 功能特点
- **3:4 写字板排版**：精确还原品牌新闻样式。
- **1+5 结构**：1张汇总封面 + 5张深度新闻内页。
- **AI 赋能**：自动抓取新闻、生成摘要，并驱动 AI 绘制配图。
- **高清输出**：支持 1800x2400 高清渲染。

## 安装方法
1. 将本项目克隆或解压到你的 Claude Skill 目录（通常是 `~/.claude/skills/green-book`）。
2. 在该目录下运行安装依赖：
   ```bash
   npm install
   ```

## 使用说明
在 Claude 界面直接输入：
> "帮我做一期关于 [主题/链接] 的小绿书"

## 资产要求
运行前请确保 `assets/` 目录下存在以下文件：
- `cover-template.png`: 封面底图模板
- `logo.png`: 品牌 Logo
